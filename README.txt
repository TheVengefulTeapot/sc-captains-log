INSTALLATION
------------
1. Run "CS Captain's Log Setup 1.0.0.exe"
2. Follow the installer prompts
3. The app will launch automatically when installation is complete
4. A desktop shortcut and Start Menu entry will be created

That's it. No other software required.


PORTABLE VERSION
----------------
If you'd prefer not to install it, "CS Captain's Log 1.0.0.exe" is a
portable version — just double-click and it runs. No installation,
no admin rights needed. You can keep it on a USB drive if you like.


YOUR DATA
---------
All data you enter is saved locally on your own machine.
Nobody else can see it, and it is not shared anywhere.

Your saved data will NOT be lost if you reinstall or update the app.
To test a clean first run, close the app and delete the relevant folder
inside %APPDATA%.

Older test builds may have used:
  %APPDATA%\cs-tracker

FIRST TIME SETUP
----------------
When you first open the app, head to Settings (top right) to:
  - Set your org name, handle and titlebar text
  - Choose a faction theme
  - Upload your org logo
  - Adjust sound and volume


UPDATING
--------
When a new version is available, just run the new installer.
It will update the app and leave your data untouched.
